Problem Statement
Financial institutions need to identify high-risk customers to reduce loan defaults and improve approval strategies.
Dataset
50K+ customer financial records
Features: income, loan amount, repayment history, credit score
Tools Used
Python (Pandas, NumPy)
EDA
Statistical Analysis
Approach
Cleaned and processed financial dataset
Performed EDA to identify risk patterns
Analyzed correlations between income, loan size, and defaults
Segmented customers based on risk levels
Key Insights
High loan-to-income ratio strongly linked to default risk
Customers with irregular repayment history showed 2x higher risk
Identified low-risk segment suitable for premium offerings
Business Impact
Supports better loan approval decisions
Helps reduce default rates
Enables targeted financial product offerings
